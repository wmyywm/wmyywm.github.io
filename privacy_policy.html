Privacy Policy for NHK Furigana Caption Viewer

Last Updated:

Data Collection

We Do NOT collect any user data including but not limited to the following data:

  • Personally identifiable information
    (e.g., name, address, email, age, ID numbers)
  • Health information
    (e.g., medical history, symptoms, diagnoses)
  • Financial/payment information
    (e.g., credit card numbers, transactions)
  • Authentication information
    (e.g., passwords, PINs, security questions)
  • Personal communications
    (e.g., emails, texts, chat messages)
  • Location data
    (e.g., GPS, IP address, nearby devices)
  • Web history
    (e.g., visited pages, browsing activity)
  • User activity
    (e.g., clicks, keystrokes, mouse movements)
  • Website content
    (e.g., private text, images, videos)

Extension Functionality

  • Only processes publicly available NHK captions to add furigana
  • All operations occur locally in your browser
  • No data is stored or transmitted to any servers

Optional Local Storage

  • May store user preferences (e.g., timing settings) locally in your browser
